Demo Program

Cubes is a graphical user interface (GUI) program that lets you communicate with your tracker using the USB (3D Guidance API) interface in an intuitive, easy-to-use way. 

You can use it to do the following:

  • Change parameters such as filter settings, measurement rate, data formats 

  • Experiment with filters

  • Alert you to nearby metal that may be affecting performance

  • Accumulate accuracy, standard deviation and noise statistics

  • Graph performance results 

  • See the effects of changing parameters

  • Log errors

  • Save/export data 

  • Obtain on-line help
